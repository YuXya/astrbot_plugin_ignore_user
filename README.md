🛡️ AstrBot 用户黑名单拦截器这是一个极其精简的插件，用于让你的 AstrBot 彻底无视某些用户的任何消息。✨ 功能特点零干扰：被拦截的用户发送消息时，机器人不会有任何回复，也不会消耗 LLM Token。高优先级：拦截逻辑运行在所有插件的最前端。简单配置：直接在 Web 管理面板添加 ID 即可。🚀 使用方法安装插件：将文件放入插件目录。获取 ID：在控制台查看你想屏蔽的用户的 ID，或者让该用户输入 /blacklist。添加黑名单：打开 AstrBot 网页管理面板。找到本插件的配置项。将 ID 添加到 黑名单用户 ID 列表 中。保存配置：立即生效，无需重启。⚙️ 技术细节使用了 event.stop_event() 机制，彻底掐断事件流。默认优先级设置为 1000，确保防火墙级别的拦截。
