# **🛡️ AstrBot 消息净化器 (黑名单 \+ 文本过滤)**

这是一个轻量级插件，用于让你的 AstrBot 无视某些用户的任何消息，或者在将消息喂给 LLM 前自动抹除特定的文字。

## **✨ 功能特点**

* **双重净化**：既能直接拦截整个用户，也能仅删除消息里的垃圾文本（如特定的前后缀标记）。  
* **零干扰**：被拦截的用户不会消耗 LLM Token；被过滤的特定文字也绝不会传给机器人。  
* **高优先级**：运行在所有插件的最前端，从源头掌控数据。  
* **简单配置**：通过 Web 管理面板即插即用，无需编写代码。

## **🚀 使用方法**

1. **获取并启用插件**：将本插件文件夹置于 data/plugins/ 下，重启 AstrBot。  
2. **获取用户 ID**：在控制台查看你想屏蔽的用户的 ID，或者让该用户对自己输入 /blacklist 查看。  
3. **配置插件**：  
   * 打开 AstrBot 网页管理面板的“插件配置”。  
   * **黑名单配置**：将 ID 写入 黑名单用户 ID 列表。  
   * **文本过滤配置**：将你想删掉的字词（如 \<正文\> 或 \</正文\>）写入 需要删除的特定文字列表 中。  
4. **保存配置**：即刻生效！

## **⚙️ 技术细节**

* **黑名单机制**：利用 event.stop\_event() 与 event.should\_call\_llm(False) 彻底掐断事件传播流。  
* **文本过滤机制**：动态修改 AstrBot 的 Plain 消息组件流，实现无感修改而不破坏消息完整性。  
* 默认优先级设为 **10000**，确保为防火墙级别的第一防线。
